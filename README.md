# Programa Hash CSV en C++ y Java
Este programa permite cargar estudiantes desde un archivo CSV y guardarlos usando una tabla hash.  
En Java se utiliza `HashMap<Integer, Estudiante>` para insertar, buscar y eliminar estudiantes por medio del `student_id`.  
También muestra el total de estudiantes cargados y evidencia el uso de una estructura hash.  
La versión en C++ trabaja la tabla hash de forma manual, mientras que Java utiliza una estructura nativa del lenguaje.
